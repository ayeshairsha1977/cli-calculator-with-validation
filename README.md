
# CLI Calculator with Input Validation

## 📌 Overview
This is a command-line calculator built using Python that performs basic arithmetic operations.  
The focus of this project is not just calculation, but handling invalid user inputs gracefully without restarting the program.

## 🚀 Features
- Supports addition, subtraction, multiplication, and division
- Handles invalid number inputs using exception handling
- Prevents program crashes due to wrong inputs
- Validates operator input
- Handles division by zero
- Continuous execution using loops (no restart needed)

## 🧠 Key Learning
This project helped me understand:
- Input validation using try-except
- Loop-based error handling
- Improving user experience in CLI applications
- Writing clean and structured logic
